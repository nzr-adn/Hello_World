Backend-first approval engine with dynamic approver resolution, idempotent API, and audit-ready workflow — without BPMN.

The Problem

Approval logic hardcoded in backend
Every change requires redeploy
Approver depends on project, role, context
Double submit & race condition bugs

The Solution

Approval Engine separates approval logic from business code. Approval flow is configuration-driven and resolved at runtime.

Flexible Approval
Sequential, parallel (ALL / ANY), conditional
Dynamic Approver
Resolver based on project, role, or owner
Backend-Safe
Idempotent API & double submit protection
Audit-Ready
Full approval history & traceability

How It Works

Client → Approval Engine API
       → Resolve approver
       → Execute approval flow
       → Return approved / rejected
